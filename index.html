Morse Code Translator

I developed a Morse code translator using Java, which utilizes a binary tree to encode and decode messages. The application allows users to input a message in morsecode and translate it into English. The binary tree data structure allows for efficient storage and retrieval of the Morse code characters, making it easy to translate large amounts of text.
